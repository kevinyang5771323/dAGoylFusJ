# 项目简介

## 核心功能
- 用户系统：注册登录、权限管理、个人中心、偏好设置
- 图书管理：信息展示、分类筛选、搜索、热门推荐
- 推荐引擎：协同过滤、内容推荐、混合推荐、相似图书
- 评论系统：评分评论、点赞举报、违规过滤
- 行为采集：浏览记录、阅读时长、无感采集
- 收藏管理：图书收藏、心愿单、优先级排序
- 数据统计：阅读分析、偏好分析、推荐效果、ECharts可视化
- 推荐定制：偏好设置、权重调整、一键刷新

## 适用场景
- 图书馆、书店、在线阅读平台
- 图书推荐、个性化阅读服务
- 教育机构、研究机构

## 技术特点
- 前后端分离，RESTful API设计
- 使用Vue 3、Django 4.2等主流技术栈
- 支持SQLite、PostgreSQL、MySQL等数据库
- 基于Scikit-learn的协同过滤算法和内容推荐算法
- 提供ECharts可视化工具进行数据统计和分析
